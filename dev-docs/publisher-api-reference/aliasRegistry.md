Exposes the aliasRegistry:

{% highlight js %}

pbjs.aliasRegistry;

or

pbjs.aliasRegistry[aliasName];

{% endhighlight %}

Can be used to fetch the entire aliasRegistry object or an individual adapter code by alias name.
